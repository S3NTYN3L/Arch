## Arch-Setup
STILL VERY MUCH A WORK IN PROGRESS!

This repo is intended for my T460 with Intel cpu and graphics.

The scripts are commented fairly well and should be pretty much self-explanatory.

Namely, it will configure the system for a completely silent boot process.

All you are presented with in the end is:

`login:`

Feel free to fork it and tweak it to suit your needs.

#### Prerequisites:
    Working Internet connection
    Logged in as `root`    
    
#### To get started:        
    curl -sL https://github.com/S3NTYN3L/Arch/archive/master.tar.gz | tar xz
        chmod u+x Arch-master/*.sh && bash Arch-master/1base.sh
